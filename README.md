# GEPT Kids English Lab

給小學生使用的英文聽力與閱讀練習工具。可直接開啟 `index.html`，不需要安裝伺服器或套件。

## 內容定位

- `小學英檢暖身 A1`：生活、校園、家庭、食物、天氣、健康等基礎情境。
- `GEPT 初級銜接 A2`：較長的日常句子、短文、告示和簡短訊息。
- 題型包含聽句選圖、閱讀理解、單字配對。
- 聽力使用瀏覽器內建 Web Speech API，美式英文語音優先。
- 練習紀錄存在瀏覽器 `localStorage`，不會上傳資料。

## 使用方式

1. 用 Chrome、Edge 或 Safari 開啟 `index.html`。
2. 選擇程度與主題。
3. 每輪練習 10 題，系統會顯示分數、正確率和連續答對。
4. 聽力題可按「字幕」核對英文原句。

## 設計依據

LTTC 小學英檢官方說明指出，小學英檢使用簡易文法概念與約 600 個基礎常用字，主題貼近臺灣小學生生活與學習經驗，程度約 CEFR A1，並可銜接 A2 的全民英檢初級。本工具依此設計為 A1 暖身與 A2 銜接，不使用官方試題原文。
